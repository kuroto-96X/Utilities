## Tap BPM ページ

### ルート

```
/bpm-tapper
```

### データモデル

タップ1回ごとに以下を記録する:

| フィールド | 型 | 内容 |
|---|---|---|
| `Index` | int | タップ番号（1始まり） |
| `Timestamp` | DateTime | タップした時刻 |
| `IntervalMs` | double? | 前タップからの間隔（ms）。初回はnull |
| `InstantBpm` | double? | 瞬間BPM = 60000 / IntervalMs。初回はnull |

### BPM計算ロジック

```
使用タップ数 N = max(1, floor(全タップ数 × スライダー値))
使用間隔     = 直近 N 個分の IntervalMs
メインBPM   = 60000 / (使用間隔の平均)
```

- スライダーは 0〜100%（デフォルト: 100%）
- タップ数が1以下の場合はメインBPMを `---` 表示

### モバイル対応

- タップボタンはビューポート高さの40%程度（`min-height: 40vh`相当）
- フォントサイズ・ボタンサイズをモバイル優先で設定
- 履歴テーブルは横スクロール対応

## サイト共通フッター

### 配置

`Layout/MainLayout.razor` の `<main>` タグ下部に `<footer>` を追加。

### 構成要素

| 要素 | 内容 |
|---|---|
| アイコン画像 | `wwwroot/icon-192.png`（既存）を流用、または別途追加 |
| 名前 | コード内直書き（外部設定ファイル不要） |
| 一言コメント | コード内直書き |
| SNSリンク | X（Twitter）など、アイコン付きリンク |
